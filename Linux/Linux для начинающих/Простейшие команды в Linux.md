## Catalog list
> Max file name lenth = 255
 
```
# files with attributres
ls -l

# root catalog
ls -l /

# home catalog
ls -l /home/

# current path
pwd

# hidden file
.file

# list with hidden files
cd ~
ls -la

# current catalog
.

# list current catalog
ls -l .
=
ls -l

# parent catalog
..

# list parent catalog
ls -la ..

# help
ls --help

```
